# WebSearch 功能说明

## 概述

WebSearch 是 Claude Code 提供的一个强大工具，允许 AI 助手搜索网络并获取最新信息。

## 主要功能

### 1. 实时信息检索
- 获取最新的技术文档
- 查找当前的最佳实践
- 搜索错误解决方案
- 获取库和框架的最新版本信息

### 2. 域名过滤

**允许特定域名：**
```
allowed_domains: ["stackoverflow.com", "github.com"]
```

**屏蔽特定域名：**
```
blocked_domains: ["example.com"]
```

## 使用场景

### 场景 1：查找最新文档
```
查询: "Python 3.12 新特性 2025"
用途: 获取最新版本的功能说明
```

### 场景 2：解决技术问题
```
查询: "React hooks useEffect 最佳实践"
用途: 查找社区推荐的解决方案
```

### 场景 3：技术选型
```
查询: "Next.js vs Remix 2025 对比"
用途: 了解不同技术栈的优劣
```

## 注意事项

1. **地域限制**: WebSearch 目前仅在美国可用
2. **时效性**: 适合查找 Claude 知识截止日期（2025年1月）之后的信息
3. **准确性**: 搜索结果会被格式化为搜索结果块返回

## 示例用法

### 基础搜索
```markdown
查询最新的 TypeScript 文档
```

### 带域名过滤的搜索
```markdown
只在 GitHub 和官方文档中搜索 Rust 异步编程示例
```

### 结合日期的搜索
```markdown
搜索 2025 年 AI 编程工具的最新进展
```

## 与其他工具的配合

- **WebFetch**: 用于获取特定 URL 的内容
- **Grep/Glob**: 用于本地代码搜索
- **Task Agent**: 结合使用进行复杂研究任务

## 最佳实践

1. ✅ **明确搜索意图**: 使用具体的关键词
2. ✅ **包含时间信息**: 如需最新信息，在查询中包含年份
3. ✅ **利用域名过滤**: 提高结果质量
4. ❌ **避免过于宽泛**: 过于笼统的查询效果不佳

## 示例输出

搜索结果会以结构化格式返回：
- 标题
- URL
- 摘要
- 相关性评分
